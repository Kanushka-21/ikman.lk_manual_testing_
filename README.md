# ikman.lk Manual Testing

This project contains manual testing documentation for the ikman.lk website.

## Contents

- **TEST_PLAN.md** - Testing plan and test cases
- **TEST_REPORT.md** - Test results and findings

## How to Use

1. Review the test plan to understand what needs to be tested
2. Execute the test cases
3. Document results in the test report

## About ikman.lk

ikman.lk is a classified ads platform for buying and selling items in Sri Lanka.
